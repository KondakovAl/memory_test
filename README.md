# Тренировка памяти

Приложение для тренировки: запоминание 6 цифр и сложение двух из них по позициям.

## Стек

- **Vite** — сборка
- **React** + **TypeScript**
- **TanStack Router** — маршрутизация
- **Mantine** — UI-компоненты

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173).

## Как играть

1. Нажмите **«Начать раунд»**.
2. За 3 секунды запомните 6 цифр в ряд.
3. После этого цифры скрываются, два квадрата подсвечиваются **красным**.
4. Сложите две цифры, которые стояли на этих позициях, и введите сумму.
5. Нажмите **«Проверить»** или Enter.

## Сборка

```bash
npm run build
npm run preview   # просмотр production-сборки
```
